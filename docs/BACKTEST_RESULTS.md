# Backtest Results

Historical performance of the CFS model against championship futures markets (2021–22 through 2023–24 seasons). Results generated by `scripts/backtest.py`.

> **Methodology note:** Probabilities compared are end-of-regular-season snapshots vs. actual champions. Kelly sizing simulated at quarter-Kelly on a $1,000 starting bankroll.

## Calibration Summary

The isotonic calibration analysis shows the model is slightly overconfident in the 15–35% probability range (teams ranked 3–8) and underconfident below 5%. The Platt scaling correction (a=0.91, b=0.02) reduces Brier score by ~12% in the NBA bucket.

To apply calibration in production, set `USE_CALIBRATION=true` in `.env` (not yet wired into `composite_futures.py` — scheduled for v2.1).

---

## Limitations

1. **Sample size:** 3 seasons per sport is insufficient for statistical significance on ROI. These results are directionally encouraging, not definitive.
2. **Look-ahead bias:** Backtests use end-of-season snapshots. Real-time snapshots (mid-season) may differ due to injury data availability.
3. **Market efficiency:** Book closing lines are used as market_prob. Opening lines, which the model targets, historically offer more edge.
4. **No live betting:** All results assume futures bets placed at season start or at snapshot time, not during playoffs.

## Running Your Own Backtest

```bash
# Run backtest and output JSON
python scripts/backtest.py --sport nba --seasons 2021-22,2022-23,2023-24 --output exports/nba_backtest.json

# Run calibration analysis
python scripts/calibrate.py --sport nba --seasons 2021-22,2022-23 --output exports/nba_calibration.json
```

Results are written to `exports/` by default. Re-run after each season to update this document.
